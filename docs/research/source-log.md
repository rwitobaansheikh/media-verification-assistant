# Research Source Log

Status: Draft

Use this log to keep research traceable. Add sources here when they influence project framing, MVP scope, datasets, model choices, evaluation, safety, or expert interview prep.

## How To Use This Table

- Prefer primary sources when possible.
- Add AI research outputs only as discovery aids, not as authoritative sources.
- Keep notes short; create a separate research brief when a source needs deeper analysis.
- Use "Needs verification" when a claim came from an AI tool, article, or secondary summary and still needs checking against a primary source.
- Use "MVP relevance" to explain why the source matters to this project.
